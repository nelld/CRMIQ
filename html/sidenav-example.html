SideNav Component Demo

This page demonstrates the SideNav component with various features.

Overview

The SideNav component provides a collapsible left navigation menu with the following features:

  • Collapsible desktop navigation with toggle button
  • Responsive mobile behavior with slide-in menu
  • Support for navigation groups and dividers
  • Badge indicators for counts or labels
  • Submenu support for nested navigation
  • Active state indication
  • Smooth animations and transitions

Desktop Behavior

On desktop screens (> 768px):

  • The navigation is visible by default on the left side
  • Click the toggle button (circular arrow) to collapse/expand
  • When collapsed, only icons are visible
  • The content area automatically adjusts its margin

Implementation

To use the SideNav component in your own pages:

  1. Add the container: <div id="sidenav-container"></div>
  2. Add the class to your main content: <main class="has-sidenav">
  3. Initialize with configuration (see SIDENAV_USAGE.md for details)
  4. Customize the navigation items array to match your needs
